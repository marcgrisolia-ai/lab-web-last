      How to enable macros
      !! INSTRUCTIONS !!
      1. Step 1 – Download the file and click the blue folder icon “Show in folder” to locate the template.
      2. Step 2 – Right-click the file → Properties → in the General tab, check Unblock (not in the Security tab).
      3. Step 3 – Click Apply, then OK. The template is ready to use.
      P.S. You only need to do this once. You can save a copy on your desktop to keep the original file intact.
